# DAD Agent (MVP)

DAD Agent is an autonomous integration-testing system.

## Features (MVP)
- Runtime UI action discovery
- Network + console observation
- Screenshot capture
- VS Code one-click runner
- Framework agnostic

## Requirements
- Node.js >= 18
- VS Code
- Any frontend app

## Run
1. Start your app
2. Click "Start DAD Agent" in VS Code
3. Observe JSON output + screenshots
